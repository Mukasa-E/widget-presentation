# Kigali Service Category Widget

This project demonstrates a reusable Flutter widget used in the Kigali City Services Directory mobile application.

## Widget Features
- Reusable category card
- Custom icon and title
- Displays number of services
- Tap interaction

## Technologies
- Flutter
- Dart

## Purpose
The widget represents service categories such as hospitals, police stations, restaurants, and parks within the Kigali Services Directory application.
